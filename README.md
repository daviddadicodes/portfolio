# David Dadi Portfolio

Premium responsive developer portfolio built with Next.js, TypeScript, and Tailwind CSS.

## Requirements

- Node.js `20.18+`
- npm `10+` recommended

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```

## Deploy to Vercel

1. Push this project to GitHub.
2. Import the repository into Vercel.
3. Keep the default Next.js framework preset.
4. Deploy.

## Project notes

- Uses real project information from `daviddadicodes` public GitHub repositories from 2026.
- Includes homepage sections, reusable components, project case-study pages, SEO metadata, and loading states.
- The resume button points to a static resume asset in `public/`.
